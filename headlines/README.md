# Headlines
Application that displays a newsfeed site that pulls in RSS feeds and displays up-to-date weather information, and currency exchange rates to the users

# Requirements
* click==6.7
* feedparser==5.2.1
* Flask==0.12.2
* itsdangerous==0.24
* Jinja2==2.9.6
* MarkupSafe==1.0
* pkg-resources==0.0.0
* Werkzeug==0.12.2

# Installation
* `pip install -r requirements.txt`
