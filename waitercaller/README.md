# Waitercaller
A Waiter Caller application that allows restaurant patrons to easily call a waiter to their table.

# Requirements
* click==6.7
* Flask==0.12.2
* Flask-Login==0.4.0
* Flask-WTF==0.14.2
* itsdangerous==0.24
* Jinja2==2.9.6
* MarkupSafe==1.0
* pkg-resources==0.0.0
* pymongo==3.4.0
* Werkzeug==0.12.2
* WTForms==2.1

# Installation
* `pip install -r requirements.txt`
